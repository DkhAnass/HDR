The MNIST database of handwritten digits, available from this page, has a training set of 60,000 examples, and a test set of 10,000 examples. Link: http://yann.lecun.com/exdb/mnist/

Testing Accuracy: 99%

## Usage 

Open the index.html page using any browser and use the input interface on the left of the page to write your handwritten digit, the CNN model will predict the written digit and showcase it on the output interface of the right of the index page.
